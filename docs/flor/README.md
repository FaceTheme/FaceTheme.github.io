# Flor

> HTML5 Video Player

## Features

- An extension and skin for video.js 7
- Video.js that is open source HTML5 & Flash video player, as well as YouTube and Vimeo (through plugins), used on over 400,000 websites
- Retina Ready & Fully Responsive
- HTML5 & CSS3
- Support and updates
- Wishlist Support
- Well Documented
